# แนวคิด UI: Interactive Investigation Graph

> ระบบกราฟโต้ตอบได้สำหรับสร้างโปรไฟล์และเชื่อมโยงบุคคล

---

## แนวคิดหลัก

ผู้ใช้สามารถ **ลากและเชื่อมต่อโหนด** (Drag & Connect Nodes) เพื่อสร้างภาพเครือข่ายการสืบสวนแบบเรียลไทม์

```
┌─────────────────────────────────────────────────────────────────────┐
│  🖱️ ลากโหนด         🔗 ลากเชื่อม        👆 คลิกดูรายละเอียด           │
│                                                                     │
│  ┌──────────┐           ┌──────────┐           ┌──────────┐        │
│  │  👤      │  โทรศัพท์ │  👤      │  พบที่    │  🏪      │        │
│  │ สมชาย    │◄─────────│ มิชชี่    │◄─────────│ Cafe     │        │
│  │ [CEO]    │           │ [Broker] │           │ [Scene]  │        │
│  └────┬─────┘           └────┬─────┘           └────┬─────┘        │
│       │                      │                      │              │
│       │ ถอนเงิน              │ รับเงิน              │ มี CCTV       │
│       ▼                      ▼                      ▼              │
│  ┌──────────┐           ┌──────────┐           ┌──────────┐        │
│  │  🏧      │           │  💰      │           │  📹      │        │
│  │ ATM      │           │ สมศรี    │           │ วิดีโอ   │        │
│  │ สยาม     │           │ [Victim] │           │ 48 ชม.   │        │
│  └──────────┘           └──────────┘           └──────────┘        │
│                                                                     │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ 🎛️ แถบเครื่องมือ                                            │    │
│  │ [เพิ่มบุคคล] [เพิ่มสถานที่] [เพิ่มหลักฐาน] [เชื่อมโยงอัตโนมัติ] │    │
│  └────────────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────────────┘
```

## ประเภทโหนด (Node Types)

### 1. บุคคล (Person Node)
```
┌─────────────────┐
│     👤          │
│   [รูปโปรไฟล์]  │
│                 │
│  สมชาย ใจดี    │
│  ─────────────  │
│  🏷️ ผู้ต้องสงสัย │
│  📞 081-xxx-xxxx│
│                 │
│  [👁️] [✏️] [🗑️] │
└─────────────────┘
```

**การโต้ตอบ:**
- ลากไปยังบุคคลอื่น → สร้างความสัมพันธ์
- ลากไปยังสถานที่ → บันทึกการพบเห็น
- ลากไปยังหลักฐาน → เชื่อมโยงกับเอกสาร/วิดีโอ
- ดับเบิ้ลคลิก → เปิดโปรไฟล์เต็ม

### 2. สถานที่ (Location Node)
```
┌─────────────────┐
│     🏪          │
│                 │
│  Cafe มุมไม้    │
│  ─────────────  │
│  📍 สยามสแควร์  │
│  🕐 48 ชม. CCTV │
│                 │
│  [▶️ ดูวิดีโอ]   │
└─────────────────┘
```

**การโต้ตอบ:**
- คลิก → ดูวิดีโอ/ภาพจากสถานที่
- ลากบุคคลมาไว้ที่นี่ → บันทึก "พบที่สถานที่นี้"

### 3. หลักฐาน (Evidence Node)
```
┌─────────────────┐
│     📄          │
│                 │
│  สเตทเมนต์ SCB  │
│  ─────────────  │
│  💰 2,000,000 ฿ │
│  📅 15 มี.ค. 68 │
│  🔗 3 entities   │
│                 │
│  [👁️ ดูเอกสาร]  │
└─────────────────┘
```

**การโต้ตอบ:**
- คลิก → ดูเอกสาร/วิดีโอ
- ลาก entity ที่พบไปยังบุคคล → เชื่อมโยง

### 4. เหตุการณ์ (Event Node)
```
┌─────────────────┐
│     🎯          │
│                 │
│  การประชุมลับ   │
│  ─────────────  │
│  🕐 14:45 น.    │
│  📍 Cafe มุมไม้ │
│  👥 3 คน        │
│                 │
│  [ดู Timeline]  │
└─────────────────┘
```

## การเชื่อมโยง (Connections)

### ประเภทความสัมพันธ์

| ไอคอน | ความสัมพันธ์ | รายละเอียดที่บันทึก |
|--------|-------------|---------------------|
| 📞 | โทรศัพท์ | เวลา, ความยาว, ใครโทรหาใคร |
| 💰 | โอนเงิน | จำนวน, วัน/เวลา, เลขบัญชี |
| 🤝 | พบกัน | สถานที่, เวลา, มีหลักฐานอะไร |
| 👤→👤 | แนะนำ | ใครแนะนำใครเข้าแก๊ง |
| 📍 | ปรากฏตัว | อยู่ที่สถานที่นี้เวลาไหน |

### การสร้างความสัมพันธ์

```
1. ลากจากโหนด A ไปยังโหนด B
        ↓
2. ป๊อปอัปเลือกประเภทความสัมพันธ์:
   ┌─────────────────────────┐
   │ เลือกความสัมพันธ์:       │
   │  ○ โทรศัพท์ 📞          │
   │  ○ พบกัน 🤝             │
   │  ○ โอนเงิน 💰           │
   │  ○ อื่น ๆ...            │
   └─────────────────────────┘
        ↓
3. กรอกรายละเอียดเพิ่มเติม
        ↓
4. บันทึก → เส้นเชื่อมปรากฏบนกราฟ
```

## โหมดการทำงาน

### 1. โหมดสำรวจ (Explore Mode)
- ดูกราฟทั้งหมด
- ซูมเข้า/ออก
- คลิกดูรายละเอียด
- กรองตามประเภท (แสดงเฉพาะบุคคล, ซ่อนหลักฐาน, ฯลฯ)

### 2. โหมดสร้าง (Build Mode)
- ลากโหนดจากแถบเครื่องมือมาวาง
- ลากเชื่อมโยงระหว่างโหนด
- แก้ไขตำแหน่งโหนด
- จัดกลุ่มอัตโนมัติ (auto-layout)

### 3. โหมดสืบสวน (Investigate Mode)
- เลือกโหนดหนึ่งเป็นจุดศูนย์กลาง
- ระบบไฮไลต์เส้นทางที่เกี่ยวข้อง
- ซ่อนโหนดที่ไม่เกี่ยวข้อง
- แสดงเฉพาะที่เชื่อมโยงภายใน 2-3 ก้าว

## ฟีเจอร์เสริม

### Auto-Connect (เชื่อมโยงอัตโนมัติ)
```
ระบบแนะนำความสัมพันธ์ที่อาจมี:

⚡ พบความสัมพันธ์ที่เป็นไปได้:
   ┌────────────────────────────────────┐
   │ สมชาย และ มิชชี่                  │
   │ ปรากฏในวิดีโอเดียวกัน (Cafe)       │
   │ เวลา 14:45 น.                      │
   │                                    │
   │ [เชื่อมโยงอัตโนมัติ] [ข้าม]         │
   └────────────────────────────────────┘
```

### Timeline Overlay
```
ลากแถบเลื่อนเวลา → กราฟเปลี่ยนตามช่วงเวลา:

14:00 ────── 14:30 ────── 14:45 ────── 15:00
  │            │            │            │
  ▼            ▼            ▼            ▼
สมศรีโอนเงิน  สมชายถอนเงิน   ทั้ง 3 พบกัน  สมชายออก

(โหนดที่ยังไม่เกิดขึ้นจางลง)
```

### Search & Highlight
```
ค้นหา: "สมชาย"
        ↓
โหนด "สมชาย" ไฮไลต์สีส้ม
เส้นเชื่อมที่เกี่ยวข้องไฮไลต์สีแดง
โหนดอื่นจางลง (fade out)
```

## ตัวอย่างการใช้งานจริง

### สร้างคดีตั้งแต่เริ่มต้น

```
Step 1: สร้างโหนดบุคคล
   → ลาก "สมชาย" มาวาง
   → ลาก "มิชชี่" มาวาง  
   → ลาก "สมศรี" มาวาง

Step 2: สร้างโหนดสถานที่
   → ลาก "Cafe มุมไม้" มาวาง
   → ลาก "ATM สยาม" มาวาง

Step 3: อัปโหลดหลักฐาน
   → ลาก "สเตทเมนต์" มาวาง → อัปโหลด PDF
   → ลาก "CCTV" มาวาง → อัปโหลดวิดีโอ

Step 4: เชื่อมโยง
   → ลาก สมชาย → มิชชี่ → เลือก "โทรศัพท์" 
   → ลาก สมชาย → ATM → เลือก "ถอนเงิน"
   → ลาก สมชาย → Cafe → ระบบแนะนำ "พบที่นี่"

Step 5: จัดเรียง
   → ลากโหนดเรียงตาม Timeline
   → หรือกด Auto-layout ให้ระบบจัดให้
```

## ข้อดีของการใช้กราฟโต้ตอบ

| ข้อดี | ตัวอย่าง |
|-------|----------|
| **เข้าใจเร็ว** | มองภาพรวมคดีใน 10 วินาที |
| **ค้นพบความสัมพันธ์** | ลากเชื่อมแล้วเห็น pattern |
| **นำเสนอได้** | โชว์ผู้บังคับบัญชา/ศาล |
| **ทำงานร่วมกัน** | หลายคนแก้ไขกราฟเดียวกัน |
| **ไม่พลาด** | เห็นทุก connection บนจอ |

## เทคโนโลยีที่ใช้ (แนะนำ)

- **React Flow** — ไลบรารี่สร้างกราฟโต้ตอบ
- **D3.js** — วาดกราฟขั้นสูง
- **Cytoscape.js** — กราฟเครือข่ายขนาดใหญ่
- **vis-network** — ง่าย เร็ว มี animation
